# CTF Pwn 题解代码库

你好！👋 我是一名专注于 **二进制漏洞利用（Pwn）** 的安全爱好者。

本仓库是我在各类CTF比赛中解出的Pwn题目的**代码档案库**。每个目录包含：

*   **题目原始二进制文件**
*   **我编写的漏洞利用脚本（EXP）**

## 📖 关于详细解题报告

我通常会将每道题的**详细分析过程、漏洞原理和完整的解题思路**整理成文章，发布在我的个人博客上。

在每个题目的目录中，你都会找到一个 `README.md` 文件，其中包含指向对应博客文章的链接。这样设计的目的是：

*   将 **GitHub** 作为纯净的**代码仓库**，方便快速查看和复现利用代码。
*   将 **个人博客** 作为深度的**分析平台**，呈现完整的技术细节和思考过程。

## 🚀 快速导航

你可以直接浏览上方的文件夹结构，找到感兴趣的题目。进入任意题目的文件夹，查看其中的 `README.md` 文件以获取：
1.  题目的基本信息。
2.  指向我个人博客上**完整Writeup**的链接。
3.  利用脚本的使用方法。
